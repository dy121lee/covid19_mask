안티코로나 캡슐 개인정보 처리 방침


안티코로나 캡슐은(는) 개인정보보호법에 따라 이용자의 개인정보 보호 및 권익을

보호하고 개인정보와 관련된 이용자의 고충을 원활하게 처리할 수 있도록 다음과 같은

처리방침을 두고 있습니다.


1. 개인정보의 처리 목적


안티코로나 캡슐은 다음의 목적을 위하여 개인정보(내 위치)를 처리하고 있으며, 다음 목적

이외의 용도로 이용하지 않습니다. 만약 다음의 목적 이외의 용도로 이용 목적이 변경될 시

사전 동의를 구할 예정입니다.


- 공공데이터 포털에서 제공하는 공적마스크 판매처 및 재고 현황 조회 API 호출
  (https://8oi9s0nnth.apigw.ntruss.com/corona19-masks/v1/storesByGeo)


2. 개인정보의 제3자 제공에 관한 사항

- 안티코로나 캡슐은 다음과 같이 개인정보(내 위치)를 제 3자에게 제공하고 있습니다.


<한국정보화진흥원 공공데이터포털>

  개인정보를 제공받는 자 : 한국정보화진흥원 공공데이터포털 (https://www.data.go/kr/)

  제공받는 자의 개인정보 이용 목적 : 공적마스크 판매처 및 재고 현황 조회 API 서비스

  제공받는 자의 보유/이용기간 : 바로 파기
